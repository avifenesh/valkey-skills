## Summary

**Bug Found:**
- **File:** `src/cluster_legacy.c`
- **Function:** `clusterShouldDeferEpochBump()`
- **Line:** 2475

**The Problem:**
The code incorrectly called `dictRelease(di)` instead of `dictReleaseIterator(di)` when releasing a dictionary iterator. This caused memory corruption and prevented the epoch collision resolution mechanism from working properly, leading to split-brain conditions after network partition recovery.

**The Fix:**
Changed line 2475 from `dictRelease(di);` to `dictReleaseIterator(di);`
