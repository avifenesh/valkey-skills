### Deploy flow
```
./deploy.sh           # ~10 min first run (valkey-search compile), fast after
./test.sh             # requires local valkey-cli with TLS support
```

> **Note:** The Docker build clones `main` of `valkey-io/valkey-search` and compiles it (~10 min). You can pin to a specific release tag by setting `--build-arg VALKEY_SEARCH_REF=v1.x.x` once the project publishes stable releases.
