## 📝 **Note on Vector Search Module**

The valkey-search module is configured in the Helm values but requires a custom Docker image (standard Bitnami image doesn't include it). The test script includes framework for vector search testing and documents this limitation. For full vector search support, you'd need to build a custom Valkey image with the module included.

All files are ready to use! Just ensure you have `kind`, `kubectl`, and `helm` installed, then run `./deploy.sh`.
